J avascript: what is the deal with Javascript?

Why is javascript so popular with developers?
 Every browser is a javascript interpreter. Javascript givens dynamic behavior to the CSS-styled html document rendered by the browser. This allows a website to provide an interactive user experience.

What are some of Javascript's limitations?
 When javascript runs in the browser it is 'sandboxed'--it can't access the files on your computer. This constraint is for security reasons. If, when you visited a website, the script they gave your browser to run could read or write data on your computer, it would be super easy for hackers to steal all your shit or totally take over your computer.
This isn't really a limitation of javascript however, although this is a common misconception. It's not even a feature of javscript per se, it's a feature of the browser as interpreter; and it's not a flaw, it's a design feature.
But javascript can run outside of the browser. One case is if hackers find a way to trick the browser into letting the javascript out of the security 'sandbox'--then allowing it run. This is known as a cross-site scripting (XSS) attack.
Another example of javascript outside the browser is node.js, which, well really i don't know what it does... framework stuff? but you can also just run scripts from the command line, analogously to using irb to run ruby code.
In node, many of the objects that are present in the javascript environment of the browser are not there, for example 'window' and 'document' are undefined.

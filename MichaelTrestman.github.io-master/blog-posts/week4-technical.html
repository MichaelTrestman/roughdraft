Ruby's Enumerable#map
 this is a shnazzy little method that 'maps' an input array onto an output array. it can be a sleeker alternative to each, and has all of the same power. that's because 'maps' takes a block as a parameter. calling map on an array returns an array of the results of calling the specified block on the items in the original.
say you have an array of hashes containing info about your favorite bands, and you want an array of the years their best albums came out. You could do like

years_of_best_album = favorite_bands.map{|band| band.best_album.release_date.year}

at least if, for each band, you've defined a best_album property with a release_date property which is a time object (or otherwise has a 'year' property). yup
